Projeto


House-Brackmann

Visão Computacional Classificação

Links

  • Apresentação TDC - The Developer Conference

Descrição

A Paralisia Facial Periférica (PFP) pode causar complicações diversas para os pacientes que a possuem, como dificuldades para alimentação ou limitações para se expressar. Além disso, por conta da pandemia do Sars-CoV-2, o diagnóstico da PFP torna-se ainda mais difícil com a necessidade do paciente de se deslocar para uma clínica médica.

O objetivo do projeto é a estimação do índice de House-Brackmann (HB) de um paciente que apresenta PFP, para fins de apoio à tomada de decisão médica. Este índice indica o grau de severidade da doença.

O projeto contou com a orientação de uma médica especialista na área de paralisia facial durante todo o desenvolvimento

Análise de Crédito

Classificação Pipelines

Links

  • Github

Descrição

A análise de crédito permite que uma instituição defina o valor liberado de um cartão de crédito, taxas de juros, quantidade de parcelas de pagamento de um produto, etc.

Este projeto tem como objetivo realizar a analise do risco de crédito para clientes de uma loja

O projeto conta com uma API cognitiva que se comunica com um modelo de machine learning moldado no padrão de projeto do Sklearn baseado em Pipelines, tornando o modelo altamente estruturado, escalável e de fácil manutenção. Para a visualização da API, foi utilizada a biblioteca Gradio. Todos os elementos se comunicam entre si via requisições em docker.

Para preservar informações confidenciais, o código foi restruturado para demonstração, além de ser utilizado um dataset público para tal objetivo

Descrição

A predição das condições finais dos materiais após usinagem é de extrema importância devido à sua experimentação ser usualmente lenta, cara e baseada em tentativa e erro

Este projeto possui como objetivo realizar a previsão da rugosidade superficial de materiais metálicos após o processamento de fresamento.

Projeto


GerBot

Processamento de Linguagens Naturais Chatbot

Links

  • Github
  • Template Front-End

Descrição

Muitos estudantes de língua alemã costumam relatar dificuldades em praticar a escrita do idioma, não somente devido à dificuldade de encontrar um(a) parceiro(a) para praticar, mas também pela insegurança em cometer erros ao conversar com uma pessoa real, o que gera lacunas de aprendizagem neste contexto.

Assim, desenvolvi uma aplicação que interage com o aluno iniciante em alemão, além de realizar correções dos erros mais comuns aos iniciantes, como gêneros dos artigos masculinos, femininos e neutros, declinações nominativas, acusativas e dativas, conjugação de verbos e erros ortográficos, além de identificar possíveis erros de concordância, como inserir um objeto inanimado como país de origem

Foi aplicado assim o uso de técnicas de processamento de linguagem natural usando a biblioteca SpaCy, e empregado Deep Learning para o funcionamento do chatbot. O modelo foi encapsulado em uma API cognitiva, e um template front-end foi utilizado para demonstrar a aplicação

Projeto


Dados Abertos de Importações

Processamento de Linguagens Naturais Classificação

Links

  • Base de Dados: Siscori - Dados Abertos da Receita

Descrição

Os dados abertos da receita permite a consulta de um conjunto de informações sobre produtos brasileiros importados, com os seus NCMs e a descrição dos produtos

Para a empresa que realiza importações, o monitoramento dessas informações, bem com a sua classificação, pode ser imprescendivel para a gestão destes produtos. Desta forma, anteriormente cada produto era manualmente classificado em planilhas eletrônicas, o que demandava muito tempo e esforço por parte do operador

Por isso, foi desenvolvida uma API cognitiva com a finalidade de realizar a classificação automática de produtos importados através do processamento de linguagens naturais e Deep Learning
