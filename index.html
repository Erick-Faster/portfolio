Projeto


House-Brackmann

Computer vision Classification

Links

  • Presentation - The Developer Conference (TDC)

Description

Problem: Peripheral Facial Palsy (PFP) can cause various complications for patients who have it, such as difficulties in feeding or limitations in expressing themselves. In addition, due to the Sars-CoV-2 pandemic, diagnosing PFP becomes even more difficult with the patient's need to travel to a medical clinic.

Solution: The aim of the project was to estimate the House-Brackmann (HB) index of a patient with PFP, for the purpose of supporting medical decision-making. This index indicates the degree of disease severity. The project was guided by a specialist doctor in the field of facial paralysis throughout its development.

Projeto


Product Classification

Natural Language Processing Classification

Links

  • Open Database: Siscori - Dados Abertos da Receita

Description

Problem: The manual classification of imported products in electronic spreadsheets is time-consuming and requires a lot of effort from the operator.

Solution: A cognitive API was developed to automatically classify imported products through natural language processing and Deep Learning. This API uses the open revenue data to access a set of information about Brazilian imported products and classify them according to their HS codes and product descriptions.

Result: For companies that import, monitoring this information and its automatic classification through the API can be essential for effective product management.
